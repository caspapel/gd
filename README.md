# Gestor de Documentos

Aplicación Spring Boot para gestionar documentos: subir, listar y descargar.

## Requisitos

- Java 21
- Maven (o usar Maven Wrapper)

## Ejecutar

```bash
./mvnw.cmd spring-boot:run
```

## Rutas

- `/` - Inicio
- `/addDocument` - Subir documento
- `/downloadDocument` - Listar y descargar documentos
- `/hola` - API de prueba
